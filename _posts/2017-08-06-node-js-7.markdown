Node.js 는 npm^Node Package Manager^을 기반으로 모듈을 공유한다.
### 외부 모듈 설치 명령어
```
$ npm install 모듈명
```

> NOTE_  
> ejs 모듈, jade 모듈 (웹 관련 모듈) 설치  
> ```
> $ npm install ejs
> $ npm install jade
> ```

> NOTE_ ejs 모듈, jade 모듈 (웹 관련 모듈) 사용
> ```
> var ejs = require('ejs');
> var jade = require('jade');
> ```
